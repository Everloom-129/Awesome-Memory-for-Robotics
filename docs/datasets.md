## Dataset Format

For each dataset, we aim to include:

- **Name**: Dataset name
- **Description**: Brief overview of the dataset
- **Size**: Number of samples, episodes, or data points
- **Modalities**: Sensor data types (vision, proprioception, etc.)
- **Tasks**: Supported tasks or use cases
- **Access**: Download links and access information
- **Citation**: Reference to the original paper
- **License**: Usage terms and conditions
